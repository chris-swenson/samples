Samples and resume of a creative data scientist experienced in full stack analytics.

# Definitions 
* Presentations: Information shared verbally, typically with an accompanying slide deck or paper. Recordings are rare.
* Technical: Code or documentation of processes, sometimes graphical or descriptive. Proprietary code cannot be shared but can be described.
* Analyses: Applications of statistical or machine learning models. Most work in this area is proprietary and cannot be shared.
* Papers: Descriptive writings that can include Analyses. Papers may be simply descriptive or may go into analytical detail.
* Reports: Graphical displays of data, typically without much interpretive text. Proprietary reports are either templates or contain simulated data.
